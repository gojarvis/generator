#Jarvis Generators
##Still under heavy development. Use with caution.
A generator for [Yeoman](http://yeoman.io).


## Getting Started

```
$ npm install -g yo
```

To install generator-jarvis from npm, run:

```
$ npm install -g generator-jarvis
```

Initiate the generator:

```
$ yo jarvis
```
